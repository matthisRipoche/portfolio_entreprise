QRScan

Présentation du projet :

Dans le cadre de ce projet réalisé avec mon entreprise Studio Stack, j’ai développé une application mobile en React Native avec Expo, connectée à une API conçue par mon patron pour le back-end. L'objectif principal était de créer une application permettant de scanner des codes-barres afin d'enregistrer la présence d'invités à un événement. Ces informations étaient ensuite transmises au serveur de Page Educ.

Contexte :

Page Educ souhaitait une solution moderne et efficace pour gérer la présence des participants lors de leurs événements. L'application devait être intuitive, rapide et capable d'interagir avec une autre entité : Page des Libraires.

Processus de réalisation :

  • Mise en place de l'environnement de développement avec Xcode et Android Studio.

  • Développement des différentes pages avec Expo Router pour permettre la navigation.

  • Connexion à l'API développée en interne pour gérer l'authentification.

  • Intégration du module de scan de codes-barres et gestion des erreurs.

  • Affichage et synchronisation des données en temps réel avec le serveur.

  • Intégration de Page des Libraires dans l'application avec exactement les mêmes fonctionnalités.

  • Tests sous Android et iOS et corrections des bugs pour une meilleure stabilité.

Conclusion :

Ce projet a été une véritable immersion dans le développement mobile, étant ma première expérience avec React Native et la création d’une application mobile. J'ai pu apprendre à structurer une application, gérer des interactions avec une API et optimiser l'expérience utilisateur. Ce défi m’a permis de monter en compétences rapidement et d’acquérir une solide base pour de futurs projets mobiles.
